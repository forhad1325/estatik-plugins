# Estatik Plugins

Custom WordPress plugins built to extend the Estatik Real Estate plugin.

Each plugin lives in its own subfolder with a dedicated README.

## Plugins

| Folder | Description |
|--------|-------------|
| [plat-map](./plat-map/) | Creates interactive plot maps for Estatik Real Estate listings |

## Requirements

- WordPress 5.0+
- Estatik Real Estate Plugin
